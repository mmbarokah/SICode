B. HTML CSS 
                            
                                <!DOCTYPE html>
                                <html>
                                <head>
                                <style>
                                body {background-color: yellow;}
                                h1 {color: teal;}
                                p {color: magenta;}
                                </style>
                                </head>
                                <body>
                                    <h1>Judul</h1>
                                 <p>Isi Paragraf.</p>
                                </body>
                                </html>
                            
                        
                    
                    CSS internal digunakan untuk menentukan gaya untuk satu halaman HTML.
                    CSS internal didefinisikan di bagian halaman HTML, dalam elemen <head><style>
                    Contoh berikut mengatur warna teks Judul menjadi teal dan teks Isi Paragraf menjadi magenta. 
                    Selain itu, halaman akan ditampilkan dengan latar belakang "kuning" warna: <h1><p>

Properti CSS yang digunakan untuk mengubah warna teks adalah...
A. text-color
B. color
C. font-color
D. text-style
