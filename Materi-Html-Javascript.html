HTML Javascript

JavaScript adalah bahasa pemrograman yang memungkinkan Anda menambahkan interaktivitas, logika, dan dinamika pada halaman web. Dengan JavaScript, Anda bisa membuat hal-hal seperti pop-up, form yang dapat divalidasi secara real-time, animasi, dan pengambilan data dari server tanpa memuat ulang halaman.an suatu website yang menarik, saling terhubung satu dengan yang lainnya, dan yang pasti dapat diakses melalui internet.

A. Struktur HTML Javascript 
                                    
                                    <!DOCTYPE html>
                                    <html>
                                    <body>
                                    <h1>My First JavaScript</h1>
                                    <button type="button"
                                    onclick="document.getElementById('demo').innerHTML = Date()">
                                    Click me to display Date and Time.</button>
                                    <p id="demo"></p>
                                    </body>
                                    </html>
                                    
                                
                    
                    Deklarasi mendefinisikan bahwa dokumen ini adalah dokumen HTML5 <!DOCTYPE html>
                    Elemen adalah elemen akar dari HTML halaman <html>
                    Elemen berisi informasi meta tentang Halaman HTML <head>
                    Elemen menentukan judul untuk HTML (yang ditampilkan di bilah judul browser atau di tab halaman) <title>
                    Elemen mendefinisikan document, dan merupakan wadah untuk semua konten yang terlihat, seperti judul, paragraf, gambar, hyperlink, tabel, daftar, dll. <body>
                    Elemen mendefinisikan judul besar <h1>
                    Elemen mendefinisikan paragraf <p>

Apa yang dilakukan oleh metode getElementById() dalam JavaScript
A. Mengambil elemen HTML berdasarkan Id-nya
B. Mengubah Id dari sebuah elemen
C. Membuat elemen HTML baru
D. Menghapus elemen HTML berdasarkan Id-nya
